üîç How it works with multiple sheets

When you upload an Excel file with multiple sheets, our comparison tool:

  • Automatically detects all available sheets in the Excel file
  • Shows a dropdown selector to choose which sheet to use
  • Uses the first sheet by default
  • Shows a notification with information about all available sheets
  • Allows you to switch between sheets without re-uploading the file
  • Updates the preview table instantly when you select a different sheet
  • Logs detailed information to the browser console
