  • Support complet des feuilles/onglets Excel
  • Sélection de feuilles spécifiques pour la comparaison

🔒 100% Traitement dans le Navigateur

  • Traitement des fichiers entièrement dans le navigateur
  • Aucun octet envoyé au serveur
  • Vos données ne quittent jamais l'appareil
  • Fonctionne complètement hors ligne

🌐 Reconnaissance Intelligente des Formats

  • Reconnaît différents délimiteurs CSV (`;`, `,`, `\t`)
  • Détecte automatiquement l'encodage des fichiers
  • Traite sans couture différents formats de fichiers
  • Aucune configuration manuelle requise

📅 Traitement Avancé des Dates

  • Reconnaît : 01-Mai-2025, 01.01.25

  • Unifie 17 et 17.0 au même format

  • Assure des comparaisons précises de nombres

⚙️ Logique de Comparaison Avancée

  • Convertit tout le texte en majuscules pour la comparaison
  • Correspondance insensible à la casse
  • Exclusion de colonnes spécifiques de la comparaison
  • Règles de comparaison flexibles

Comment Ça Marche et Sécurité

🔒 100% Traitement Côté Client

Tout le traitement des fichiers se fait directement dans votre navigateur en utilisant JavaScript. Aucune donnée n'est jamais envoyée à un serveur.

  • Les fichiers ne sont stockés que dans la RAM du navigateur
  • Aucun traitement côté serveur
  • Aucun stockage cloud ou base de données

🛡️ Sécurité des Données

Vos données confidentielles sont entièrement sécurisées grâce à plusieurs couches de protection :

  • Aucune requête réseau avec les données des fichiers
  • Nettoyage automatique à la fermeture de la page
  • Aucun journalisation ou analytique du contenu

⚡ Comment le Traitement Fonctionne

Comparaison de fichiers avancée utilisant les technologies web modernes :

  • API FileReader pour l'accès local aux fichiers
  • Bibliothèque XLSX.js pour les fichiers Excel
  • Algorithmes intelligents de correspondance floue

🔍 Cycle de Vie des Données

Transparence complète sur la façon dont vos données sont traitées :

  • Téléchargées → Traitées → Affichées
  • Automatiquement nettoyées au rechargement
  • Aucun stockage persistant nulle part
