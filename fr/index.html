Comment Ça Marche et Sécurité

🔒 100% Traitement Côté Client

Tout le traitement des fichiers se fait directement dans votre navigateur en utilisant JavaScript. Aucune donnée n'est jamais envoyée à un serveur.

  • Fichiers stockés uniquement dans la RAM du navigateur
  • Aucun traitement côté serveur
  • Aucun stockage cloud ou base de données

🛡️ Sécurité des Données

Vos données sensibles sont complètement sûres avec plusieurs couches de protection :

  • Aucune requête réseau avec les données de fichier
  • Effacé automatiquement à la fermeture de la page
  • Aucun logging ou analytics du contenu

⚡ Comment Fonctionne le Traitement

Comparaison avancée de fichiers utilisant les technologies web modernes :

  • API FileReader pour l'accès aux fichiers locaux
  • Bibliothèque XLSX.js pour les fichiers Excel
  • Algorithmes de correspondance floue intelligents

🔍 Cycle de Vie des Données

Transparence complète sur la façon dont vos données sont traitées :

  • Chargé → Traité → Affiché
  • Effacé automatiquement au rechargement
  • Aucun stockage permanent nulle part
