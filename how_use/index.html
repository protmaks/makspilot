How to compare two Excel files or Excel and CSV

1. Open the site MaksPilot in any modern browser on your computer.

2. Select two files, Excel or CSV; if there are several tabs in Excel, select the one you need. Currently, there is a limit on the number of rows and columns because the browser resources are limited, but if the project is in demand, then it is possible to increase the file sizes for comparison.

3. Several lines from the file are displayed under the file to make sure you have selected the required one.

4. Click to Compare.

5. At the top is the final pivot table, which displays the number of lines in each file, the number of lines that match and do not match, as well as the columns that are in one file but not in the other.

6. Below is a field where you can enter a list of columns and exclude them from the comparison.

7. Below we see a table with detailed results. The lines are grouped, and the differences are highlighted, which allows you to quickly find discrepancies.

The orange lines are those that are only in one table. The red ones are those that have differences; they are grouped, and the green ones show what is similar. The white ones are those that are present in two tables.

8. There is a filter under the heading of each column.
