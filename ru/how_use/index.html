Как сравнить два Excel файла или Excel и CSV

1. Откройте сайт MaksPilot в любом современном браузере на вашем компьютере.

2. Выберите два файла, Excel или CSV; если в Excel несколько вкладок, выберите нужную. В настоящее время есть ограничение на количество строк и столбцов из-за ограниченности ресурсов браузера, но если проект будет востребован, то можно увеличить размеры файлов для сравнения.

3. Под файлом отображается несколько строк из файла, чтобы убедиться, что вы выбрали нужный.

4. Нажмите "Сравнить".

5. Сверху находится итоговая сводная таблица, которая отображает количество строк в каждом файле, количество совпадающих и не совпадающих строк, а также столбцы, которые есть в одном файле, но нет в другом.

6. Ниже находится поле, где можно ввести список столбцов и исключить их из сравнения.

7. Ниже мы видим таблицу с подробными результатами. Строки сгруппированы, а различия выделены, что позволяет быстро найти несоответствия.

Оранжевые строки - это те, которые есть только в одной таблице. Красные - те, в которых есть различия; они сгруппированы, а зеленые показывают, что похоже. Белые - те, которые присутствуют в двух таблицах.

8. Под заголовком каждого столбца есть фильтр.
